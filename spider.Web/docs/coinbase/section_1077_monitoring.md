# Monitoring
Source: https://docs.cdp.coinbase.com/get-started/cdp-portal/monitoring



When you use API keys associated with your project in the CDP Portal, you can view a set of default metrics and visualizations in your project Dashboard. Use these metrics to better understand and continue iterating on your application.

<info>
  Metrics do not update during your session, but refreshing the page reloads the page with the most up to data available.
</info>

## Time Periods and Increments

You can set the time period with the time range filter in the top right. All times are calculated in UTC and presented in the user’s local timezone.

* **Last 30 days:** The last fully completed 30 days.
  * Bar and line charts present data in fixed increments of 24 hours (12:00 AM -11:59 PM).
* **Last 7 days:** The last fully completed 7 days.
  * Bar and line charts present data in fixed increments of 8 hours (12:00AM - 7:59AM, 8:00AM - 3:59PM, 4:00PM - 11:59PM).
* **Last 24 hours:** The last fully completed 24 hours.
  * Bar charts present data in fixed increments of 60 mins.
  * Line charts present data in fixed increments of 30 mins.
