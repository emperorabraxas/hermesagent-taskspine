## CDP AI integrations

**Context Menu** - Quick access via "Copy page" dropdown on any docs page

* Direct AI tool integration (ChatGPT, Claude, Perplexity)
* One-click IDE setup (Cursor, VS Code)

**MCP Server** - Ongoing AI tool access to CDP documentation

* Works with Claude, Cursor, VS Code, and other MCP-compatible tools
* Enables searchable access to the entire documentation library
