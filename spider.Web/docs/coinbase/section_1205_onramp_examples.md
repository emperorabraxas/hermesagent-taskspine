# Onramp Examples
Source: https://docs.cdp.coinbase.com/onramp/additional-resources/examples



Experience everything Onramp has to offer by [downloading and running a sample app](/get-started/demo-apps/starter/onramp-demo-app):

## Sample web app

1. Clone the [demo web repo](https://github.com/coinbase/onramp-demo-application).
2. [Make a secret API key](/onramp/introduction/quickstart#prerequisites) and move it into the repo as `api_keys/cdp_api_key.json`.
3. Follow the instructions in the README to run the app.

## Sample mobile app

1. Clone the [demo mobile repo](https://github.com/coinbase/onramp-v2-mobile-demo).
2. Follow the instructions in the README to run the app.
