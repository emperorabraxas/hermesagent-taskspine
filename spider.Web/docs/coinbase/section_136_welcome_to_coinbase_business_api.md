# Welcome to Coinbase Business API
Source: https://docs.cdp.coinbase.com/api-reference/business-api/rest-api/introduction



## Introduction

The Coinbase Business API provides a comprehensive suite of tools for businesses to integrate cryptocurrency functionality into their applications. This API reference section contains detailed documentation for business-first APIs, while general business APIs are documented in the [Coinbase Business documentation](/coinbase-business/introduction/welcome).

<Note>
  To use these APIs, you need a [Coinbase Business account](https://www.coinbase.com/business).
</Note>

## API Documentation Structure

### Business-First APIs

Business-first APIs are designed specifically for business use cases and have their complete API reference documentation in this section:

* **[Checkouts API](/api-reference/business-api/rest-api/checkouts/introduction)**: Create and manage checkouts for accepting cryptocurrency payments. Generate embeddable checkout URLs with real-time status tracking and webhook notifications.

### General Business APIs

For general business APIs that are shared across multiple Coinbase products, please refer to the main [Coinbase Business documentation](/coinbase-business/introduction/welcome):

* **[Advanced Trade APIs](/coinbase-business/advanced-trade-apis/overview)**: Programmatic trading and order management
* **[Transfer APIs](/coinbase-business/transfer-apis/send-crypto)**: Send crypto, manage addresses, and handle fiat operations
* **[Track APIs](/coinbase-business/track-apis/accounts)**: Monitor accounts, track transactions, and access market data

## Getting Started

Before using these API references, please visit the [Coinbase Business Getting Started guide](/coinbase-business/introduction/get-started) to:

* Set up your account and API keys
* Understand authentication requirements
* Learn about our API architecture and best practices

Once you've completed the getting started guide, you can explore the API references in this section for business-first APIs or visit the [Coinbase Business documentation](/coinbase-business/introduction/welcome) for general business APIs.
