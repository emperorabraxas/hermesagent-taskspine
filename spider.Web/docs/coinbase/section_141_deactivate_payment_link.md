# Deactivate Payment Link
Source: https://docs.cdp.coinbase.com/api-reference/business-api/rest-api/payment-links/deactivate-payment-link

post /api/v1/payment-links/{paymentLinkId}/deactivate
**Deprecated.** Use the Checkouts API instead. Deactivates a payment link, preventing it from accepting further payments. Only `ACTIVE` payment links can be deactivated.
