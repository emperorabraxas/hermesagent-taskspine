# Get Bookable Block Trade Accounts
Source: https://docs.cdp.coinbase.com/api-reference/derivatives-api/rest-api/block-trades/get-bookable-block-trade-accounts

/api-reference/derivatives-api/rest-api/cde-spec.json get /rest/block-trade/booking/bookable-accounts
Retrieves the list of block trade accounts that the authenticated user can book trades for.
