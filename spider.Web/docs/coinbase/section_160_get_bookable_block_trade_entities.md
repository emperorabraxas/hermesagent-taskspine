# Get Bookable Block Trade Entities
Source: https://docs.cdp.coinbase.com/api-reference/derivatives-api/rest-api/block-trades/get-bookable-block-trade-entities

/api-reference/derivatives-api/rest-api/cde-spec.json get /rest/block-trade/booking/bookable-entities
Retrieves the list of block trade entities that the authenticated user can book trades for.
