# Create Zero Firm Product Group Limit
Source: https://docs.cdp.coinbase.com/api-reference/derivatives-api/rest-api/firm-product-group-limits/create-zero-firm-product-group-limit-1

/api-reference/derivatives-api/rest-api/cde-spec.json post /rest/v2/firm-product-group-limit
Creates a zero position limit to a specific firm/product group that disables trading for that combination.
