# SwapTransactionEvmEoaData
Source: https://docs.cdp.coinbase.com/sdks/cdp-sdks-v2/frontend/@coinbase/cdp-hooks/Interfaces/SwapTransactionEvmEoaData

| Property                | Type                                                                                            | Description                                                                                                                                                                                                                |
