# Modal
Source: https://docs.cdp.coinbase.com/sdks/cdp-sdks-v2/frontend/@coinbase/cdp-react/Components/Modal



```ts theme={null}
const Modal: FC<DialogProps> = Dialog;
```

The Modal component is the same as the Dialog component from Radix UI. This component is used to render the Radix UI Dialog component as a modal.
