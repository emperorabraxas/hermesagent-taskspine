# ModalClose
Source: https://docs.cdp.coinbase.com/sdks/cdp-sdks-v2/frontend/@coinbase/cdp-react/Components/ModalClose



```ts theme={null}
const ModalClose: ForwardRefExoticComponent<DialogCloseProps & RefAttributes<HTMLButtonElement>> = DialogClose;
```

The ModalClose component is the same as the DialogClose component from Radix UI. It is intended to be used with the [Modal](/sdks/cdp-sdks-v2/frontend/@coinbase/cdp-react/Components/Modal) component.
