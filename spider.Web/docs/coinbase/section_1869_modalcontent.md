# ModalContent
Source: https://docs.cdp.coinbase.com/sdks/cdp-sdks-v2/frontend/@coinbase/cdp-react/Components/ModalContent



```ts theme={null}
const ModalContent: ForwardRefExoticComponent<ModalContentProps & RefAttributes<HTMLDivElement>>;
```

The ModalContent component is a wrapper around a DialogOverlay and DialogContent. It is intended to be used with the [Modal](/sdks/cdp-sdks-v2/frontend/@coinbase/cdp-react/Components/Modal) component.
