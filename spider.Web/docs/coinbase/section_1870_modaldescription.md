# ModalDescription
Source: https://docs.cdp.coinbase.com/sdks/cdp-sdks-v2/frontend/@coinbase/cdp-react/Components/ModalDescription



```ts theme={null}
const ModalDescription: ForwardRefExoticComponent<DialogDescriptionProps & RefAttributes<HTMLParagraphElement>> = DialogDescription;
```

The ModalDescription component is the same as the DialogDescription component from Radix UI. It is intended to be used with the [Modal](/sdks/cdp-sdks-v2/frontend/@coinbase/cdp-react/Components/Modal) component.
