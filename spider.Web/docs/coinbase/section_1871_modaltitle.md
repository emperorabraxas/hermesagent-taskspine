# ModalTitle
Source: https://docs.cdp.coinbase.com/sdks/cdp-sdks-v2/frontend/@coinbase/cdp-react/Components/ModalTitle



```ts theme={null}
const ModalTitle: ForwardRefExoticComponent<DialogTitleProps & RefAttributes<HTMLHeadingElement>> = DialogTitle;
```

The ModalTitle component is the same as the DialogTitle component from Radix UI. It is intended to be used with the [Modal](/sdks/cdp-sdks-v2/frontend/@coinbase/cdp-react/Components/Modal) component.
