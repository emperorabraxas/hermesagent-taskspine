# Unlock Trading
Source: https://docs.cdp.coinbase.com/api-reference/derivatives-api/rest-api/risk/unlock-trading

/api-reference/derivatives-api/rest-api/cde-spec.json post /rest/unlock-trading
Deactivates kill switch to cancel orders and unlocks trading for specified users, firms, or FCMs. Must only include one uuid (fcm, firm, or trading user).
