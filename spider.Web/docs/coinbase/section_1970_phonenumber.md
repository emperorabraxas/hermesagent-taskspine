# PhoneNumber
Source: https://docs.cdp.coinbase.com/sdks/cdp-sdks-v2/frontend/@coinbase/cdp-react/Interfaces/PhoneNumber



A phone number object.

## See

[parseValuesFromPhoneNumber](/sdks/cdp-sdks-v2/frontend/@coinbase/cdp-react/Utilities/parseValuesFromPhoneNumber) for the function that parses a phone number into a phone number object.

## Properties

| Property      | Type     | Description                                                                    |
| ------------- | -------- | ------------------------------------------------------------------------------ |
| <a /> `value` | `string` | The phone number in the national format (country calling code is not included) |
| <a /> `e164`  | `string` | The phone number in international format (E.164). Typically used in API calls. |
