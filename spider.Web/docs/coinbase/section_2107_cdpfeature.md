# CdpFeature
Source: https://docs.cdp.coinbase.com/sdks/cdp-sdks-v2/frontend/@coinbase/cdp-solana-standard-wallet/Type-Aliases/CdpFeature



```ts theme={null}
type CdpFeature = {
  cdp:: true;
};
```

Custom feature to distinguish CDP Solana Wallet from other standard Solana wallets.

This feature flag allows dapps to identify CDP wallets specifically.

## Properties

| Property     | Type   |
| ------------ | ------ |
| <a /> `cdp:` | `true` |
