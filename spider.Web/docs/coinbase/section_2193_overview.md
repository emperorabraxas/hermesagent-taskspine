# Overview

CDP Wallets are secured by a [Wallet Secret](/server-wallets/v2/introduction/security), which can only be accessed by the wallet owner/developer. Wallet Secrets are asymmetric key pairs used to authenticate with a [Trusted Execution Environment](/server-wallets/v2/introduction/security). If you lose your Wallet Secret, or you suspect it has been compromised, it is crucial to rotate it promptly to ensure safety of your funds. This action can be performed via the CDP portal.
