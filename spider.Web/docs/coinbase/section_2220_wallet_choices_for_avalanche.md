# Wallet choices for Avalanche
Source: https://docs.cdp.coinbase.com/staking/staking-delegation-guides/avalanche/delegation-guide/wallet-choices-for-avalanche



Currently, staking is available using [Avalanche Wallet](https://wallet.avax.network/).

You can also stake using Avalanche Wallet and a Ledger hardware device. For details, see [How to use a Ledger Nano S or Nano X with Avalanche?](https://docs.avax.network/learn/setup-your-ledger-nano-s-with-avalanche/#how-to-set-up-avalanche-on-ledger-live).
