# Cardano Delegation Guide
Source: https://docs.cdp.coinbase.com/staking/staking-delegation-guides/cardano/cardano-delegation-guide



[Cardano](https://cardano.org) is a proof of stake blockchain based in peer-reviewed research and focused on security, scalability, and financial applications. You can learn more about Cardano [in our protocol overview](https://www.coinbase.com/learn/crypto-basics/what-is-cardano).

| Protocol Parameter      | Value                                                       |
| :---------------------- | :---------------------------------------------------------- |
| Minimum tokens to stake | 10 ADA                                                      |
| Warmup period           | 2 Epochs (10 days)                                          |
| Unbonding period        | Instant                                                     |
| Service fee             | 5% of rewards + fixed fee of 340 ADA (spread among stakers) |
