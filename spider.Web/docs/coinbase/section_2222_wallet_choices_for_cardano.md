# Wallet choices for Cardano
Source: https://docs.cdp.coinbase.com/staking/staking-delegation-guides/cardano/wallet-choices-for-cardano



Currently, staking is available using the [Daedalus Mainnet desktop wallet](https://daedaluswallet.io) or the [AdaLite web wallet](https://adalite.io).

Additionally, you may delegate using the CLI as outlined in [Cardano's documentation](https://docs.cardano.org/stake-pool-operators/creating-keys-and-certificates).
