# Celestia Delegation Guide
Source: https://docs.cdp.coinbase.com/staking/staking-delegation-guides/celestia/celestia-delegation-guide



[Celestia](https://celestia.org) is a modular consensus and data network that makes it easy for anyone to deploy a blockchain with minimal overhead.

Users can secure the network by delegating to a validator and receive a share of its staking rewards.

| Protocol Parameter | Value   |
| :----------------- | :------ |
| Minimum to stake   | 1 TIA   |
| Warm-up period     | None    |
| Unbonding period   | 21 Days |
| Service fee        | 10%     |

<Warning>
  **Service fee**\
  Coinbase may change its service fee during the lifetime of this validator (e.g., increasing fees as total delegation increases to ensure we are never running an outsized portion of the network on our node).
</Warning>
