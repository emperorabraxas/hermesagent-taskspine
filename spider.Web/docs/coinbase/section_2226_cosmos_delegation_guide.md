# Cosmos Delegation Guide
Source: https://docs.cdp.coinbase.com/staking/staking-delegation-guides/cosmos/delegation-guide/cosmos-delegation-guide



The [Cosmos Network](https://cosmos.network) is a decentralized network of independent, scalable, and interoperable blockchains. You can learn more about the protocol in our [Cosmos Guide](https://www.coinbase.com/learn/crypto-basics/what-is-cosmos).

<video />

Coinbase public validator offers high-uptime, and is architected to minimize downtime and double-signing. You retain full control over your assets when you delegate with self-custody wallets on Coinbase.

| Protocol Parameter      | Value         |
| :---------------------- | :------------ |
| Minimum tokens to stake | 1 ATOM        |
| Warmup period           | None          |
| Unbonding period        | 21 days       |
| Service fee             | 8% of rewards |
