# Wallet choices for Cosmos
Source: https://docs.cdp.coinbase.com/staking/staking-delegation-guides/cosmos/delegation-guide/wallet-choices-for-cosmos



Currently, staking is available using [Ledger Live](https://www.ledger.com/ledger-live) and a Ledger Nano S or X hardware device, and also by using any one of a variety of web-based wallets, including [Keplr](https://wallet.keplr.app/#/dashboard).

In addition, staking is available using any wallet that interacts with Cosmos command line tools, such as a Ledger wallet or file-based signer. If you need help staking using the Cosmos CLI, see the Cosmos [Command-Line Interface](https://docs.cosmos.network/v0.46/core/cli.html) docs.
