# Staking ATOM with Ledger Live
Source: https://docs.cdp.coinbase.com/staking/staking-delegation-guides/cosmos/ledger-live/staking-with-ledger-live



[Ledger Live](https://www.ledger.com/ledger-live) is a mobile and desktop application that works in tandem with a Ledger hardware device to manage digital assets held in cold storage.

If you do not have any ATOM stored on your Ledger device, start at [1: Set up and fund Ledger Live wallet](/staking/staking-delegation-guides/cosmos/ledger-live/set-up-and-fund-ledger-live-wallet). If you already have ATOM stored on your Ledger device, proceed to [2: Stake to with Ledger Live](/staking/staking-delegation-guides/cosmos/ledger-live/stake-to-with-ledger-live).
