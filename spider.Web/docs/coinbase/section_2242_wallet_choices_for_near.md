# Wallet choices for Near
Source: https://docs.cdp.coinbase.com/staking/staking-delegation-guides/near/delegation-guide/wallet-choices-for-near



Currently, staking is available using the [NEAR CLI](https://docs.near.org/tools/near-cli) or [NEAR web wallet](https://wallet.near.org/), and using a Ledger Nano S or X hardware device.

If you have a NEAR token allocation, such as from Coinlist or the NEAR Foundation, or if you recently purchased your NEAR from an exchange, the NEAR web wallet is currently the easiest method to delegate from your lockup contract.

The NEAR CLI commands can be used if you are delegating from an account or a lockup contract.
