# Staking with NEAR CLI
Source: https://docs.cdp.coinbase.com/staking/staking-delegation-guides/near/near-cli/staking-with-near-cli



In order to stake, you must install the Node.js Pack Manager and [NEAR CLI](https://github.com/near/near-cli) to perform the tasks listed below.
All commands can be run in the Terminal on your computer.

<Info>
  You may choose to delegate to either of the Coinbase Stake Pools: `bisonstrails.poolv1.near` or
  `bisonstrails2.poolv1.near`.

  For the purposes of clarity only the `bisonstrails.poolv1.near` address is included in these directions. Those wishing to stake to the `bisonstrails2.poolv1.near` pool should replace all instances the address appears with `bisonstrails2.poolv1.near`.
</Info>
