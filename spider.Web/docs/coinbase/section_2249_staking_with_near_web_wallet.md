# Staking with NEAR Web Wallet
Source: https://docs.cdp.coinbase.com/staking/staking-delegation-guides/near/web-wallet/staking-with-near-web-wallet



The [NEAR web wallet](https://wallet.near.org/) is a non-custodial web-based wallet that allows you to interact with NEAR accounts.

If you do not have any NEAR stored on your NEAR web wallet wallet, start at [1: Set Up and Fund NEAR Web Wallet](/staking/staking-delegation-guides/near/web-wallet/set-up-and-fund-near-web-wallet). If you already have NEAR stored on your NEAR web wallet, or if you have a NEAR web wallet address provided at your token allocation (ending in .near) proceed to [2: Stake to with NEAR Web Wallet](/staking/staking-delegation-guides/near/web-wallet/stake-with-near-web-wallet).
