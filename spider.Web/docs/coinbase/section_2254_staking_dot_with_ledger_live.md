# Staking DOT with Ledger Live
Source: https://docs.cdp.coinbase.com/staking/staking-delegation-guides/polkadot/ledger-live/staking-with-ledger-live



Ledger Live is a mobile and desktop application that works in tandem with a Ledger hardware device to manage digital assets held in cold storage.

If you do not have any DOT stored on your Ledger device, start at [1: Set Up and Fund Ledger Live with DOT](/staking/staking-delegation-guides/polkadot/ledger-live/setup-and-fund-ledger-live). If you already have DOT stored on your Ledger device, proceed to [2: Stake DOT to Coinbase with Ledger Live](/staking/staking-delegation-guides/polkadot/ledger-live/stake-with-ledger-live).
