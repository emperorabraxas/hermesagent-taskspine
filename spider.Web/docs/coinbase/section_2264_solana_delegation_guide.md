# Solana Delegation Guide
Source: https://docs.cdp.coinbase.com/staking/staking-delegation-guides/solana/delegation-guide/solana-delegation-guide



Solana is a fast, secure, and scalable blockchain network for distributed apps, marketplaces, and enterprise applications.

By encoding the passage of time as data, Proof of History (PoH) provides a sense of trustless time and message ordering before consensus, reducing messaging overhead between nodes in the network and enabling greater overall scalability. For more, see our [Guide to Solana](https://www.coinbase.com/cloud/discover/protocol-guides/guide-to-solana).

| Protocol Parameter      | Value         |
| :---------------------- | :------------ |
| Minimum tokens to stake | 1 SOL         |
| Warm up period          | 2 days        |
| Unbonding period        | 2 days        |
| Service fee             | 8% of rewards |
