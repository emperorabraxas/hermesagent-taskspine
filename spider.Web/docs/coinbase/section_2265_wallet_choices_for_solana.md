# Wallet choices for Solana
Source: https://docs.cdp.coinbase.com/staking/staking-delegation-guides/solana/delegation-guide/wallet-choices-for-solana



Currently, staking is available on the browser-based [Phantom wallet](https://phantom.app/), or on [SolFlare](https://solflare.com/) with a Keystore File or Ledger Nano S or X hardware wallet device.

In addition, staking is available using any wallet that interacts with Solana command line tools. Before you begin with the CLI, please be sure you have created a stake account (different than a wallet address) in a supported wallet. If you need help creating a stake account, see the Solana docs on [Staking](https://docs.anza.xyz/cli/examples/delegate-stake).
