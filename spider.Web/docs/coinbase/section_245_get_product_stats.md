# Get product stats
Source: https://docs.cdp.coinbase.com/api-reference/exchange-api/rest-api/products/get-product-stats

GET /products/{product_id}/stats
Gets 30day and 24hour stats for a product.

<Info>
  The`volume` property is in base currency units. Properties `open`, `high`, `low` are in quote currency units.
</Info>
