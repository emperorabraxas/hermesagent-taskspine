# Withdraw to payment method
Source: https://docs.cdp.coinbase.com/api-reference/exchange-api/rest-api/transfers/withdraw-to-payment-method

POST /withdrawals/payment-method
Withdraws funds from the specified `profile_id` to a linked external payment method

<Info>
  **Withdraw funds to a payment method**

  See the [Payment Methods](/api-reference/exchange-api/rest-api/transfers/get-all-payment-methods) section for retrieving your payment methods.
</Info>

## API Key Permissions

This endpoint requires the "transfer" permission. API key is restricted to the default profile.
