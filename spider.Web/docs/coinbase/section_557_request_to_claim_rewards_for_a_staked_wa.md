Source: https://docs.cdp.coinbase.com/api-reference/prime-api/rest-api/staking/request-to-claim-rewards-for-a-staked-wallet

post /v1/portfolios/{portfolio_id}/wallets/{wallet_id}/staking/claim_rewards
Request to claim staking rewards. This feature is in alpha. Please reach out to your Coinbase Prime account manager for more information
