# Add an EVM account to an end user
Source: https://docs.cdp.coinbase.com/api-reference/v2/rest-api/end-user-accounts/add-an-evm-account-to-an-end-user

post /v2/end-users/{userId}/evm
Adds a new EVM EOA account to an existing end user. End users can have up to 10 EVM accounts.
This API is intended to be used by the developer's own backend, and is authenticated using the developer's CDP API key.
