# Prepare and send a user operation for EVM Smart Account
Source: https://docs.cdp.coinbase.com/api-reference/v2/rest-api/evm-smart-accounts/prepare-and-send-a-user-operation-for-evm-smart-account

post /v2/evm/smart-accounts/{address}/user-operations/prepare-and-send
Prepares, signs, and sends a user operation for an EVM Smart Account. This API can be used only if the owner on Smart Account is a CDP EVM Account.
