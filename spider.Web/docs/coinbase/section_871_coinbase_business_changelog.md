# Coinbase Business Changelog
Source: https://docs.cdp.coinbase.com/coinbase-business/introduction/changelog



These release notes list upcoming and recent changes to the Coinbase Business API.

## 2025

### 2025-DEC-03

* **Invoicing API**: Added new Invoicing API for retrieving and reading invoice data for Coinbase Business accounts.

### 2025-NOV-27

* **Payment Links Webhooks**: Added webhook support for Payment Links API, enabling real-time notifications for payment status changes (success, failed, expired).

### 2025-OCT-31

* **Payment Links API**: Added new Payment Links API for creating and managing cryptocurrency payment links.

### 2025-JUN-30

* **Published Coinbase Business APIs**: Trade, Transfer, and Track API's for Coinbase Business accounts.
