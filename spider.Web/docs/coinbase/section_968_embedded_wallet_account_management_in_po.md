# Embedded Wallet: Account Management in Portal
Source: https://docs.cdp.coinbase.com/embedded-wallets/account-manager



<Tags />

## Overview

The Account Management dashboard in CDP Portal provides a centralized view to monitor all embedded wallets created by your users.

## Accessing the dashboard

Access the Account Management dashboard directly on [CDP Portal](https://portal.cdp.coinbase.com/products/embedded-wallets/accounts).

## Features

The dashboard provides comprehensive visibility into your users' wallets:

<Frame>
  <img alt="Account Management dashboard showing user accounts" />
</Frame>

<CardGroup>
  <Card title="User Accounts" icon="users">
    View all your users' accounts with addresses and creation dates in a sortable table.
  </Card>

  <Card title="Account Details" icon="wallet">
    Click into any account to view balances and policies.
  </Card>
</CardGroup>

Click into any wallet to see address, creation date, token balances, and user info.

<Frame>
  <img alt="User account details showing balances and policies" />
</Frame>

For additional insights, access the [metrics dashboard](https://portal.cdp.coinbase.com/products/embedded-wallets/metrics).

## Use Cases

<CardGroup>
  <Card title="Monitor User Activity" icon="chart-simple">
    View wallet usage patterns to understand how your application is performing.
  </Card>

  <Card title="Debug User Issues" icon="bug">
    Quickly look up a specific user's wallet to investigate transaction failures or balance discrepancies.
  </Card>

  <Card title="Track Balances" icon="coins">
    Monitor aggregate balances across all user wallets to understand total value secured by your application.
  </Card>

  <Card title="Compliance and Reporting" icon="file-chart-column">
    Export wallet lists and activity data for compliance reporting or internal analytics.
  </Card>
</CardGroup>
